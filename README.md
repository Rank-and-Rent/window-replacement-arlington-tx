# Home Window Replacement Service of Arlington

A modern, SEO-optimized website for a window replacement business serving Arlington, TX and the Dallas-Fort Worth metroplex.

## Features

- **Premium Design**: Elegant dark theme with high-contrast typography inspired by Weather Shield
- **SEO Optimized**: Rich semantic HTML, Schema.org structured data, and keyword-focused content
- **Responsive**: Mobile-first design that works on all devices
- **Performance**: Built with Next.js 14 for optimal loading speeds
- **Animations**: Smooth scroll animations with Framer Motion

## Window Types Featured

- Double-Hung Windows
- Single-Hung Windows
- Casement Windows
- Awning Windows
- Sliding Windows
- Picture Windows
- Bay & Bow Windows
- Special Shape Windows

## Brands

- Andersen
- Pella
- JELD-WEN

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- CSS Modules
- Framer Motion

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000)
